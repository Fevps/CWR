# CWR
Experimental, overpowered, lightweight, open-sourced Content Warning cheat/hacks.

![Menu](https://raw.githubusercontent.com/Fevps/CWR/main/ContentWarning%20Menu/Properties/Menu.png)
This client is the best looking available and most up-to-date Content Warning menu.
It includes multiple tabs/categories of a variety of different modules/cheats.

### This menu is still in development, any issues please text @functionkey3 on Discord.

# Installation
- Download the latest [release](https://github.com/Fevps/CWR/releases/tag/CWR)
- Download [MelonLoader 0.7.1](https://melonloader.co/#download)
- Install MelonLoader for Content Warning

![Melon](https://raw.githubusercontent.com/Fevps/CWR/main/ContentWarning%20Menu/Properties/Melon.png)

- Drag the release (.dll) to Mods after importing MelonLoader

![Mods](https://raw.githubusercontent.com/Fevps/CWR/main/ContentWarning%20Menu/Properties/Mods.png)

- Now load the game and enjoy (might have to disable anti-virus.)

# How To Use
- Right-Shift to toggle visibility of the GUI
- Use right-click/middle-mouse-button to drag the GUI
- Left-Click to use press UI elements (buttons, etc.)
- Navigate through the categories and click the checkbox on a module
